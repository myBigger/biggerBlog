最近为了存放 Parallels 虚拟机的文件，我在闲鱼买了一块金士顿 A400 480G 固态硬盘。卖家说是全新的，宣称读写速度能达到 530MB/s。

结果一测，速度根本跑不满。

## 问题一：速度跑不满

| 指标 | 官方宣传 | 实际测试 |
| ---- | ------- | ------- |
| 读取速度 | 500MB/s | 350MB/s |
| 写入速度 | 450MB/s | 280MB/s |

用 Blackmagic Disk Speed Test 测了好几次，基本都是这个结果。一开始我还以为是固态硬盘翻车了，结果查了一圈发现问题不在硬盘，而是我的**拓展坞**：

- 拓展坞是 USB 5Gbps（理论带宽 500MB/s）
- USB 接口有带宽损耗，实际能用到 350MB/s 就不错了
- 这还是只接一个硬盘的情况

**结论：不是固态的问题，是我的设备接口限制了速度。**

## 问题二：固态硬盘罢工

后来我把机械硬盘也接上拓展坞，金士顿固态直接罢工了——识别不到，磁盘工具里也找不到。

**原因：供电不足。**

- USB 接口供电能力有限
- 机械硬盘启动时电流大
- 两个硬盘同时接，供电跟不上了
- 固态硬盘直接不干了

## 解决思路

**方案一：只接固态（目前我在用）**

固态硬盘单独接拓展坞，机械硬盘不用时就拔掉。

**方案二：换硬盘盒**

如果需要同时接多个硬盘，可以考虑：
- 带独立供电的硬盘盒
- 或者直接接电源的 USB Hub

## 总结

买固态硬盘前，务必确认以下几点：

1. **接口带宽**：USB 3.0/5Gbps → 最高 350MB/s；USB 10Gbps/雷电 → 才能跑满
2. **供电能力**：确认拓展坞/硬盘盒能否同时供电多个设备
3. **实际测试**：不要相信卖家的宣传，自己测一测

这次算是花钱买教训了。希望看到这个文章的各位，少走弯路。
